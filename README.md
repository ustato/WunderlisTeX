# WunderlisTeX
WunderlistのタスクをLaTeXで可視化できるようにする


# Introduction

手軽なタスク管理ツールを利用したくなり，更にはその結果がLaTeX製資料に掲載できれば嬉しいと考えた．
そのアイデアを形にしたアプリケーションである．

[Wunderlist](https://www.wunderlist.com)が提供しているAPIを利用し，タスクのリスト毎に表スタイルのタスク管理表と，一週間内に行ったタスクを可視化できる．
